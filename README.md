codiff
======

> [codiff manpage](doc/codiff.md)  
> [comparison of color diffs](doc/comparison.md)  

Color diff that prints old and new lines with changed characters/words/blocks highlighted.  
It streams the GNU diff output line by line, coloring it on the fly.  

<a href=doc/codiff.png><img width=640px src=doc/codiff.png></a>

### INSTALL:

The [codiff](codiff) is a standalone Perl script, just copy or download it and run.  
Tested on Linux, should work on macOS too.

Direct link: https://raw.githubusercontent.com/rudojaksa/codiff/main/codiff
